# Knowledge Injection

## Setup

### Environment

This codebase uses Python 3.7.9. Other versions may work as well.

Dependencies:

```
$ conda create -n ki -y python=3.7.9
$ conda activate ki
(ki) $ pip install -r requirements.txt
```

## Running experiments
From the root dir, run the command below:
```
(ki) $ python experiments/gpt_ft.py
```
